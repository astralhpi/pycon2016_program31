# Neovim 소개
## 말그대로 vim의 미래
vim을 적극적으로 리팩토링한 superset

???
공식 사이트의 메인에 있는 슬로건을 그대로 옮긴다면, neovim은 '말그대로 vim의 미래'
입니다. neovim은 vim을 적극적으로 리팩토링한 프로젝트로, 기존 vim에서 기능이
추가된 superset을 지향하고 있습니다.
